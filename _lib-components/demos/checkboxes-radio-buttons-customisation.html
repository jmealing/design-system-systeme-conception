Option Description How to use
Disabled state Disabled inputs will not be clickable. Visual indications include greying out the control and adding a 'stop' cursor icon on hover.
  • Add the class is-disabled to the list item of the checkbox or radio item
  • Add a disabled attribute to the input field
Inline radio buttons Inline radio buttons or checkboxes will display side by side on larger screens and wrap to new lines, rather than vertically. Add the class form-inline to the fieldset or parent element of the checkbox ul.
